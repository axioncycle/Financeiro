# Axion Cycle — Gestão de Performance

App de gestão financeira para operações de e-commerce de rodas de carbono.

## Funcionalidades

- **Dashboard** — KPIs, gráficos de linha interativos, meta mensal, capital em estoque
- **Vendas** — Registro com cálculo automático de taxa, imposto, comissão e lucro
- **Estoque** — Sistema de contêineres com distribuição por unidade e FIFO
- **Resultado** — DRE mensal com despesas fixas e extras
- **Custo** — Planejamento de contêiner, rateio de câmbio, break-even e ROI

## Stack atual

- HTML + CSS + JavaScript ES5 puro
- localStorage (temporário — migrando para Supabase)
